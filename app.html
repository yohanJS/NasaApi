NasaApp

Mars Rover Photos

This API is designed to collect image data gathered by NASA's Curiosity, Opportunity, and Spirit rovers on Mars and make it more easily available to other developers, educators, and citizen scientists. This API is maintained by Chris Cerami.

I am a big fan of The Martian. Both the Book and the Movie, so I thought it was cool to use Nasa API to find all the photos taken on Mars. The web app is simple to use. Images are organized by the sol (Martian rotation or day) on which they were taken. A photo taken on Curiosity's 1000th Martian sol exploring Mars, for example, will have a sol attribute of 1000. Input a number that is greater than 0, and the application will retrieve all the images taken on that Martian Sol.
